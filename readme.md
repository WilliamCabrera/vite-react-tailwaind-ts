# Vite React Tailwind Template

This template provide toolchain below:

- Vite
- React, ReactDOM
- ESLint
- Prettier
- Git Hook (Husky, Lint Staged)
- TailwindCSS 3

# Usage

```sh
pnpm i
pnpm dev
```
